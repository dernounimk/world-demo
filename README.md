# 🌍 Dernouni World

My personal corner on the web – a space where I share my thoughts on life, books, games, hobbies, and stories. This website is a static demo that showcases my front-end development skills through a responsive, visually appealing layout built with pure HTML, CSS, and a touch of JavaScript.

🌐 **Live Demo**: [dernounimk.github.io/world-demo](https://dernounimk.github.io/world-demo/)

---

## ✨ Features

- 📝 **Personal Blog / Articles** – Read my posts on topics like "Cat's Love", "Dream of a Child", "Video Games", and more.
- 🖼️ **Gallery Section** – A visual display of images (content to be added by you).
- 💪 **Skills Showcase** – Highlights my proficiency in **HTML, CSS, JavaScript, and Python**.
- 📊 **Event Countdown** – A dynamic timer for "Tech Masters Event 2024".
- 💰 **Pricing Plans** – Three responsive pricing tiers (Basic, Advanced, Professional).
- 🎥 **Top Videos List** – A curated list of tutorial and gaming videos.
- 📈 **Statistics Counter** – Visual stats for years of experience, clients, projects, and awards.
- 🎨 **Custom & Responsive Design** – Unique CSS styling that works on all devices.
- ⚡ **Lightweight & Fast** – No heavy frameworks or libraries.
- 🚀 **CI/CD Ready** – Includes a GitHub Actions workflow (`static.yml`) for automated deployment to GitHub Pages.
